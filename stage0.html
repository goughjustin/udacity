How it works

Computers communicate over the web by sending a HTTP Request to a server. the server sends back the correct web page document to the users browser. This video does a good job showing you how the web works.


HTML

HTML Documents make up the majority of the Web. HTML stands for Hyper Text Markup Language. This language is how the web is coded.


Elements - Inline vs Block

Elements are either inline or block. The block elements have an invisible block around them. You can adjust the height and width of the invisible block.
