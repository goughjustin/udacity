Lesson 1: The Basics of the Web and HTML

How it works

Computers communicate over the web by sending a HTTP Request to a server. the server sends back the correct web page document to the users browser. This video does a good job showing you how the web works.

HTML

HTML Documents make up the majority of the Web. HTML stands for Hyper Text Markup Language. This language is how the web is coded.

Elements - Inline vs Block

Elements are either inline or block. The block elements have an invisible block around them. You can adjust the height and width of the invisible block.

Lesson 2: Structured Document in HTML

Developer Tools

Google Chrome has great developer tools. Click the menu in the top right corner, select more tools and then select developer tools. You can quickly access the Developer Tools by right click and clicking, Inspect Element.

The DOM

HTML uses a tre like structure called the DOM. You can draw lines between each element in the DOM.

HTML is just Boxes!

HTML content is just boxes. When you see the PDF layout, you can asses how many elements may be needed by using a pencil to outline each box.

Text Editors

Writing code us much easier if you are using a text editor like Sublime Text 2. The editor helps you visually see where each element is indented (nested).

Lesson 3: Adding Style with CSS

Thing to Avoid

When coding you want to avoud repition. Repition leads to copy and pasting styles and being unable to change later.

What is CSS

CSS stands for Cascading Style Sheets. CSS allows your to control the style and look of your page from a single CSS page source. This makes making changes easy and does not require a lot of copy and pasting.

Have Fun and Enjoy adding Style!
